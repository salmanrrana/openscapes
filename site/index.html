A no-permission ambient instrument

Your browser has weather.

OpenScapes turns the quiet facts of this visit into a drifting sound field. It tries to play on load, then waits for one click if the browser blocks sound.

The composition uses page load timing, connection quality when available, CPU cores, device memory when available, screen size, clock drift, performance jitter, pointer motion, and click position.

It does not read real network requests, Wi-Fi traffic, browser history, files, or private data. These signals stay inside this page.

Click or touch anywhere after entry. Every pixel is a seed: x changes pan and pitch, y changes brightness and decay.
